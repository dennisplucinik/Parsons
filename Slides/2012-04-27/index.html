Performance Checklist

  • Network Speed Problems:
    • Reduce HTTP requests
      • image sprites
      • concatenate CSS & JavaScript
    • Serve device specific content
      • smaller images on smaller screens
      • optimize for high resolution displays
      • check connection speed [link]
  • Device Limitation Problems:
    • Fallback CSS3 transitions to jQuery animations
    • Every performance issue gets amplified

CSS3 → jQuery Fallback

/* -------CSS:------- */
#element{
  left: 0px;
  transition: all 1s;
}

#element:hover {
  left: 100px;
}


/* -------JavaScript:------- */
if(!Modernizr.csstransitions){
  $('#element').hover(
    function(){ $(this).animate({left:'100px'}), // over
    function(){ $(this).animate({left:'0px'}) // out
  });
}
