Selectors

How to connect CSS to HTML

  • Three ways to match a CSS rule to an HTML element: tag, class, or id.

Specificity

Calculating order of importance

  • Each of the three selectors has a different specificity.
  • In order from high to low: id > class > tag
  • You can chain selectors together to increase specificity.
  • Whichever rule has the highest specificity gets applied.

The Box Model

  • Defines rules for calculating height, width, margin, padding, & border
  • Provides structure for everything.

Positioning

  • All elements default to position: static (i.e., "unpositioned")
  • Position options are "static", "relative", "absolute", and "fixed".
  • Absolute vs. Relative
    • An element with either is said to be "positioned" and can take on other attributes
      (e.g., top, left, right, bottom, z-index, etc.)
    • Absolutely positioned elements are placed exactly from the top-left of the closest "positioned" parent.
    • Absolutely positioned elements are removed from the normal document flow.
    • Relatively positioned elements are placed relative to their natural position in the document flow.
